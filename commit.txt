Initial commit for beta backend release:

- Added database models representing the data schema
- Configured DbContext and added base classes implementing the Repository Pattern for the Data Access Layer (DAL)
- Added Serilog table in the database and overrode save changes
- Created repositories, defaulting the schema for all tables, and added data seeds
- Created the Web API project and added DbContext and data repositories to the DI container
- Built the Base Controller and controllers
- Updated Swagger settings and added logging to the Web API
- Added Exception Filter and Async methods for the CRUD controller and repositories
- Utilized AutoMapper and DTOs
- Added filtering, pagination, and sorting to the Base CRUD controller and dynamically constructed LINQ queries for entity-specific repositories
- Implemented Authentication and Authorization, and added CORS Policy with required auth for all endpoints by default
